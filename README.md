# FlowNAS:
This repository contains the code for the project for the Deep Learning Course at Sapienza *FlowNAS*.

All the tests were ran on two datasets: NAS201 and NAS301, although we recommend testing the code on NAS201 as NAS301 is heavy and takes longer to configure. For the purpose of making setup and testing easier this README is devided into the following sections: 
- A brief overiview of the files of the project
- Setup steps for NAS201
- Setup steps for NAS301
